Installationsguide

Lägg till Valvet i din MCP-klients konfiguration:

{
  "mcpServers": {
    "valvet": {
      "url": "https://mcp.promptbanken.se/mcp",
      "headers": { "X-MCP-Key": "DIN_NYCKEL_HÄR" }
    }
  }
}

Verktygen list_my_items, search_my_items, get_my_item fungerar direkt. save_my_item är tillgängligt för alla (Free: 5 nya/månad via MCP). update_my_item och archive_my_item kräver Pro.
