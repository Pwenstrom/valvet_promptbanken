MCP-nyckel

Din nyckel öppnar valvet direkt inifrån Claude eller ChatGPT.

Skapa en nyckel

Din nya nyckel — kopiera den nu, den visas bara en gång:

Tänk på nyckeln som en riktig nyckel till en riktig dörr. Varje AI-klient (Claude, ChatGPT) är en dörr in till ditt valv — koppla in nyckeln i klienten så kan den läsa och (om du har Pro) skriva i dina insättningar åt dig. Välj din klient nedan.

C Claude (Desktop eller webb)
  1. Öppna Claude och gå till Inställningar → Anslutningar (Settings → Connectors).
  2. Välj Lägg till anpassad anslutning och klistra in adressen https://mcp.promptbanken.se/mcp.
  3. Under anpassade headers, lägg till X-MCP-Key som namn och klistra in din nyckel som värde.
  4. Spara. Claude kan nu läsa dina insättningar via list_my_items, search_my_items och get_my_item — och spara nya via save_my_item.

Ser du inte Anslutningar i din version? Redigera claude_desktop_config.json manuellt istället — se konfigurationen nedan.

G ChatGPT
  1. Öppna ChatGPT och gå till Inställningar → Anslutningar (Settings → Connectors) — funktionen rullas ut stegvis och kräver för närvarande Plus, Pro eller Team.
  2. Välj Lägg till anpassad anslutning och ange adressen https://mcp.promptbanken.se/mcp.
  3. Lägg till en anpassad header: X-MCP-Key med din nyckel som värde.
  4. Aktivera anslutningen i en ny chatt (via verktygsikonen) för att låta ChatGPT läsa och spara i ditt valv.

Saknar du alternativet? Din plan eller version kan sakna stöd för anpassade anslutningar ännu.

{ } Manuell konfiguration (avancerat)

För MCP-klienter som konfigureras via en JSON-fil (t.ex. Claude Desktops claude_desktop_config.json):

mcpServers-block
{
  "mcpServers": {
    "valvet": {
      "url": "https://mcp.promptbanken.se/mcp",
      "headers": { "X-MCP-Key": "DIN_NYCKEL_HÄR" }
    }
  }
}

list_my_items, search_my_items och get_my_item fungerar direkt för alla. save_my_item är tillgängligt för alla (Free: 5 nya/månad via MCP). update_my_item och archive_my_item kräver Pro.
